Comparison Operators
> greater than
    ans: Checks if the left value is bigger than the right value.
< less than
    ans: Checks if the left value is smaller than the right value.
>= greater than or equal to 
    ans: Checks if the left value is greater than OR equal to the right value.
=== Strict Equal To
    ans: Checks if value AND data type are the same.
== equal to (Loose Equality)
    ans: Checks if values are equal, but ignores data type. it convert value
!== not equal to 
    ans: Checks if the values are NOT equal OR NOT the same type.
!== not equal to (loose) 
    ans: Checks if the values are NOT equal but it does NOT check the data type.
