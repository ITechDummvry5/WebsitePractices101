Syntax Rules For variable? 

ex: let v2 = 2;
Explanation:
let → creates a new variable (alternative is const)
v2 → variable name
= → assigns a value
2 → the value (number or string)
; → ends the statement

Variable name Restriction?
1. cant use variable name let 
ex: let let = 3;        
2. cant start with a number 
ex: let 2name = 3l      
3. cant use special characters 
    let  !@#$%^&*() = 2;
except Allowed special characters: $ dollarsign &  _ underscore 
    let $name = charles;
    let _age  = 14;

Assigning a value to a variable?     This is default 
Re-Assigning a value to a variable?  This is we change see  //output 5 and //output  2

Syntax Rules For Re-assigning a value?
1. Dont Use let when Re-assigning
ans: When you change the value of an existing variable, do not use let again.
NOTE! let is only used when creating the variable for the first time.
Re-assigning means you are just changing the value, not creating a new variable.
2.The variable must already exist
3.You cannot re-assign a const variable
ans: A variable created with const cannot be changed.
